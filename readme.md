本系统用到：html+css+JavaScript
所用插件为：echarts+jspanel+jstree



财政数据的json文件放在ser文件夹下
统计数据的json文件放在data文件下



Index.html文件为页面主界面
frame.html为指标树文件
Popup.html为弹窗文件



Index.js文件为所有的js文件
popupChart.js文件为弹窗js文件


需要注意的：系统中有函数drawJsTree()与函数drawTree()
            drawJsTree()为在四川省地图时绘制的指标树
            drawTree()为在是地级市地图时绘制的指标树
            drawMapPic()为绘制四川省地图并且绑定响应事件
            drawMap()为绘制地级市地图
